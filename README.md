熊猫创意开源节点 PandaOpenNode
=============

使用方法
===

-  将 upload 内的源码上传到任意支持 PHP 的空间
-  打开网址，点击链接跳转到查询界面即可

依赖的组件
====

-  CURL
-  MBString

如果还开启了 redis 或 memcache 的支持还可以实现百度异常次数和查询次数的监控
